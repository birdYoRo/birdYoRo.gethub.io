Introduction

机器学习是怎么找函式 Regression输出数值，Classification做出选择

  • Regression回归 ： the output of the function is a scalar //输出是数值

  • Binary Classification：输出是yes or no（二值输出）

  • Multi-class Classification：输出从Class1，Class2，…，Class N中选择一个输出

  • Supervised Learning：监督学习，提供给机器有label的资料（Labeled Data）学习；函式的Loss（损失函数），用Labeled Data去评价function的好坏（loss），机器来选出loss最低的function。

    //比如Alpha Go对固定的一步，有明确的下一步下法

  • Reinforcement Learning （强化学习）：

    //通过直接和自己下，得到的每一次输赢结果（Reward 来纠正学习过程）来获得下一步的下法。

  • Neural network：神经网络

  • Linear functoin：线性函数

  • Network Architecture：给机器一个函式搜寻范围

  • CNN：卷积神经网络

  • RNN：循环神经网络

有关pyenv

常用pyenv指令

  • pyenv versions 列出目前系统中所有安装的Python
  • pyenv version 表示目前预设的Python版本
  • pyenv global 设定预设Python版本
  • pyenv install 安装特定版本的Python
  • pyenv uninstall 移除特定版本的Python
